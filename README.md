# Mathieu Rivière — site personnel

Site personnel académique (humanités numériques, histoire, enquêtes), publié via GitHub Pages.

- URL: [https://icimathieu.github.io](https://icimathieu.github.io)
- Générateur: Jekyll
- Réalisé avec l'aide de Codex/ChatGPT en mode opt-out, sans partage des données de ce projet pour l'entraînement des modèles.
- Licence: Apache-2.0 (voir `LICENSE`)

## Fonctionnalités

- Navigation: Archives, Blog, CV, Projets, Publications, Recherche, Vidéos.
- Recherche interne: fonctionnelle (index JSON + recherche côté navigateur).
- Flux RSS unique: fonctionnel (blog + publications + vidéos).

## Flux RSS

Le flux RSS unique permet de s'abonner aux mises à jour des vidéos, du blog et des publications.

Pour s'abonner aux nouvelles publications du site:

1. Copier l'URL du flux unique: `https://icimathieu.github.io/feed.xml` ou cliquer sur le lien dans le footer du site.
2. Ouvrir un lecteur RSS (par exemple Feedly, Inoreader, NetNewsWire).
3. Ajouter une nouvelle source/abonnement et coller cette URL.
4. Les nouveaux contenus (blog, publications, vidéos) apparaîtront automatiquement dans le lecteur.
